# Music Web App

## Music Web App Back-End
The back-end uses errors, multiple requests (GET, POST), and a schema. You can create a user to send to the databse to be stored using MongoDB Compass. 

## Database Users

When the users are created and sent to the database. You can view it using MongoDB Compass to view what has been created to verify that the user has been created. When created you can sign in with email and password.

## Working with Back-End Errors

The errors are created within its respected file in the folder. It sends errors if the information is sent inncorrectely. For example an unauthorized request will give ""Incorrect email or password"".
